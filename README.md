Projeto de Windows Forms desenvolvido em C# para calcular a idade de uma pessoa com base no ano de nascimento. Ideal para iniciantes que querem praticar interface gráfica, eventos e lógica básica em C#.

Funcionalidades

Inserir ano de nascimento

Calcular idade automaticamente

Interface gráfica amigável e intuitiva

Tecnologias

C#

.NET Framework

Windows Forms

Visual Studio

Como usar

Abra o projeto no Visual Studio.

Compile e execute.

Insira o ano de nascimento e clique em "Calcular" para ver a idade.

Observações

Projeto desenvolvido como exercício de aprendizado.

Funcionalidade simples, mas demonstra manipulação de eventos, entradas de usuário e interface gráfica.
