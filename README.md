Calcualdora de idade.

Projeto de Windows Forms desenvolvido em C# para calcular a idade de uma pessoa com base no ano de nascimento. Ideal para iniciantes que querem praticar interface gráfica, eventos e lógica básica em C#.

Funcionalidades
Este projeto permite que o usuário insira seu ano de nascimento e, ao clicar no botão "Calcular", a aplicação exibe automaticamente a idade correspondente. A interface é simples, amigável e intuitiva, facilitando o uso para qualquer pessoa.

Tecnologias
O projeto foi desenvolvido utilizando C#, .NET Framework e Windows Forms, dentro do ambiente Visual Studio.

Como usar
Para utilizar o projeto, basta abrir o arquivo no Visual Studio, compilar e executar. Insira o ano de nascimento na caixa indicada e clique em "Calcular" para ver o resultado.

Observações
Este projeto foi criado como exercício de aprendizado e demonstra os conceitos básicos de manipulação de eventos, entrada de dados do usuário e criação de interface gráfica em C#.

AgeCalc

This is a Windows Forms project developed in C# to calculate a person's age based on their birth year. It is ideal for beginners looking to practice graphical interfaces, event handling, and basic logic in C#.

Features
The application allows users to input their birth year and, by clicking the "Calculate" button, automatically displays the corresponding age. The interface is simple, user-friendly, and intuitive, making it easy to use for anyone.

Technologies
The project was built using C#, .NET Framework, and Windows Forms within the Visual Studio environment.

How to use
To use the project, open the solution in Visual Studio, compile, and run it. Enter the birth year in the designated box and click "Calculate" to see the age.

Notes
This project was created as a learning exercise and demonstrates fundamental concepts such as event handling, user input processing, and graphical interface creation in C#.
